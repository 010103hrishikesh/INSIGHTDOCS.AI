# INSIGHTDOCS.AI

A document intelligence application that allows users to upload PDF files, ask questions in natural language, and generate context-aware answers using Retrieval Augmented Generation (RAG) and local Large Language Models.

## What this project does

This system processes unstructured PDF documents and converts them into searchable semantic embeddings.
When a user asks a question, relevant document chunks are retrieved from a vector database and passed to a local LLM to generate an answer based only on document context.

## Core Workflow

PDF Upload
Text Extraction
Text Chunking
Embedding Generation
Vector Similarity Search
Context Retrieval
LLM Response Generation

## Key Functionalities

* Upload and process PDF documents
* Semantic document search using embeddings
* Context-aware question answering
* Document summarization
* Local LLM inference using Llama3
* Dockerized backend deployment

## Running the Project

Start backend

```bash
python -m uvicorn api:app --reload
```

Start frontend

```bash
streamlit run app.py
```

Docker

```bash
docker build -t insightdocs .
docker run -p 8000:8000 insightdocs
```

## Project Structure

app.py → Streamlit frontend
api.py → FastAPI backend
ingest.py → PDF ingestion pipeline
rag_pipeline.py → Retrieval + LLM pipeline
Dockerfile → Containerization setup
